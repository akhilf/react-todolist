## Todo List App

Prerequisites
============
* node.js 
* Git 
* Clone the code from GitHub
* Take the latest pull from Git

Project setup
===============
* Go to the project folder structure
* Run -> npm install
* Run -> npm run start
* Tests -> npm test
* E2E -> npx playwright test
